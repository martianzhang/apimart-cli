apimart-cli --help
APIMart API 的统一命令行工具。支持图片生成、任务查询、余额查询等操作。

子命令：
  image     图片生成（文生图、图生图、Inpainting）
  video     视频生成（文生视频、图生视频）
  chat      AI 对话（流式输出，默认 deepseek-v4-flash）
  models    查询模型列表及定价（无需 API Key）
  task      查询任务状态
  balance   查询余额

默认值可在 ~/.config/apimart/config.yaml 中配置。

Usage:
  apimart-cli [command]

Available Commands:
  balance     Query API key or user account balance
  chat        Chat with AI models (streaming by default)
  help        Help about any command
  image       Generate images via the APIMart API
  models      List available AI models from APIMart marketplace
  task        Query task status and result
  version     Print the version information
  video       Generate videos via the APIMart API

Flags:
      --api-base string     API base URL (env: APIMART_API_BASE, default https://api.apimart.ai)
      --api-key string      APIMart API key (env: APIMART_API_KEY)
      --config string       path to config file (default ~/.config/apimart/config.yaml)
  -h, --help                help for apimart-cli
      --http-proxy string   HTTP proxy URL (env: APIMART_HTTP_PROXY, e.g. http://127.0.0.1:7890)
      --json string         JSON file path, JSON string, or "-" for stdin
      --output string       output directory for downloaded images (default ".")
      --save-prompt         save prompt to .md file alongside results
  -v, --verbose             verbose output: show full result JSON
      --version             version for apimart-cli

Use "apimart-cli [command] --help" for more information about a command.
