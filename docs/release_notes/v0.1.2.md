新功能
- 交互式多轮对话（chat）：不带 --message 自动进入 REPL 模式，连续对话，支持流式输出
- 交互模式快捷操作：/exit / quit / bye / 退出 / Ctrl+C / Ctrl+D 均可退出
- chat 支持 --verbose / -v：显示 token 消耗、费用、耗时统计
- 全局 --model / -m 标志：所有子命令统一入口，各命令自行处理默认值逻辑
破坏性变更
- image / video 不再硬编码默认模型：必须通过 --model 或 defaults.image.model / defaults.video.model 配置指定，否则报错
- flag 体系调整：
- --json → 仅 image / video 可用（非全局）
- --mode → 仅 image 可用（非全局）
- --save-prompt → 仅 image 可用（非全局）
- --output → 保留全局
修复
- 跨平台 Ctrl+D 退出：使用 golang.org/x/term 原始终端模式，Windows / Linux / macOS 均可识别
- Ctrl+C 优雅退出：捕获 SIGINT 信号，不走 os.Exit 暴力退出
文档
- 全量更新 docs/guide-chat.md / docs/guide-image.md / docs/guide-video.md
- 更新 config.example.yaml 和用户配置，移除已迁移的非全局配置项
