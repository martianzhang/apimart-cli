## ✨ OpenRouter 集成

`base_url` 指向 `openrouter.ai` 时自动适配 OpenRouter 专有 API：

### 图片生成 — 自动路由

| 模型类型 | 使用 API | 示例模型 |
|---|---|---|
| 专用图片模型 | `POST /v1/images` (Dedicated Image API) | `openai/gpt-image-2`, `openai/dall-e-3` |
| 原生图片输出模型 | `POST /v1/responses` (Responses API) | `google/gemini-3.1-flash-image-preview` |

### 视频生成 — 异步 submit → poll → download

- OpenRouter 视频 API 自动适配
- 任务持久化：提交后自动保存 job 信息，`--job-id` 重新拉取

### 模型发现 — 免认证

- `apimart-cli models --type image` → `GET /v1/images/models`
- `apimart-cli models --type video` → `GET /v1/videos/models`

## 🛡 健壮性

- HTML 响应检测：代理返回封锁页面时给出明确提示
- 全局超时从 30s 提升至 120s
- OpenRouter 图片请求使用独立 120s 超时
