## ✨ New Feature

- **`ideas` 命令** — 从 Image2Studio 公开 API 搜索 AI 图片提示词灵感，无需 API Key

  ```bash
  apimart-cli ideas "cinematic portrait"
  apimart-cli ideas "cat" --limit 10
  apimart-cli ideas "portrait" --page 2
  apimart-cli ideas "luxury" --json | jq '.results[].prompt'
  apimart-cli ideas "product" --save
  ```
