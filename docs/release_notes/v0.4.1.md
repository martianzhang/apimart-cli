## ✨ New Feature

- **`ideas` 命令** — 本地提示词灵感搜索，基于 15243 条精选提示词（NeXra + CSV 多源合并）
- **`preview` 命令** + `image --preview` / `video --preview` — 终端内嵌图片预览（Sixel/iTerm2/Kitty）
- **`ideas --save`** — 下载参考图片到本地
- **`ideas --random`** — 随机抽取（默认启用）
- **`ideas --json`** — JSON 输出，配合 jq 使用

## 🔧 改进

- `ideas` 数据嵌入二进制（go:embed），编译即可用，无需联网
- 全部 HTTP 请求统一走代理配置
- 自动构建：推送 v* tag 即触发 CI 交叉编译 + 上传 release
