# v0.6.0 — Agentic Chat

## ✨ 新功能

### Agentic Chat（聊天中调用工具）
- chat 命令支持 tool calling，LLM 可在对话中生成图片、视频等
- 9 个工具：`generate_image`、`generate_video`、`midjourney_imagine`、`midjourney_describe`、`midjourney_reroll`、`midjourney_video`、`ideas`、`balance`、`task`
- 所有工具通过共享函数实现，`chat` 与对应 CLI 命令走同一份代码

### 交互式 REPL 增强
- 快捷键：`Ctrl+A/E/U/K/W/L`、上下箭头（命令历史）
- `/<tool_name>` 自然语言调工具，LLM 自动解析参数
- `!<shell_cmd>` 执行系统命令（自动识别 SHELL 环境变量，30s 超时）
- `/tools` 列出所有工具
- `apimart-cli` 无参数时默认进入 chat 交互模式

### 新配置项
- `defaults.chat.max_iterations` — 每次用户消息 LLM 最多连续调用工具次数
- `defaults.chat.tools` / `disable_tools` — 工具白名单/黑名单（glob 模式）
- `defaults.chat.allow_tool_override` — 是否允许 LLM 覆盖配置参数（默认 false，省钱）
- `defaults.image.resolution` — 新增 resolution 代码默认值

## 🐛 Bug 修复
- 图片/视频生成强制使用用户配置（LLM 选择的昂贵参数被正确覆盖）
- `-v` verbose 模式在 root 命令下正确显示
- persistent flag 检测修复：`hasFlagChanged()` 检查 local + persistent + inherited flags
- balance URL 双拼 bug
- 管道输入正确识别为单轮模式
- 所有 `RunE` 命令补全 `SilenceUsage: true`

## 🔧 重构
- `generateImageAndSave` / `generateVideoAndSave` 共享函数，CLI 和 agent loop 统一入口
- `loadImageDefaults` / `loadVideoDefaults` 直接从文件加载配置，不依赖 shared.Cfg
- `readLineRaw` 重写，支持完整 readline 快捷键
- `config.LoadDefaults` 调用全部替换为 `shared.Cfg`
- 代码默认值改为低成本：`quality=low`、`resolution=1k`
